# Struktur
- controllers
    - Controller für die einzelnen Seiten
- Core 
    - wichtigste Classes
- public
    - index.php
- views
    - HTML-Stuff für jede Seite und aufgeteilt in die einzelnen Blöcke (head, header, footer, card, set, ...)

# Setup
- php installieren
- script starten mit `php -S localhost:8080 -t public`
- http://localhost:8080
